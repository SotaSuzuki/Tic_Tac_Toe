# Tic Tac Toe

It is a Tic tac toe game which is made by React. I reffered to [React Official Tutorial](https://facebook.github.io/react/tutorial/tutorial.html) to step into the world of React.

## Installation

```bash
npm install
```

## Usage

### Watch & Server

```bash
npm start
```

### Build

```bash
npm run build
```
